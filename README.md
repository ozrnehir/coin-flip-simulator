# 🪙 Coin Flip Simulator

An interactive coin flip (Heads/Tails) game built with HTML, CSS, and JavaScript.  
This project demonstrates core frontend concepts such as DOM manipulation, animations, and event-driven programming.

---

## 🚀 Features

- 🎲 Random coin flip logic  
- 🔄 Smooth CSS animation  
- 📊 Real-time statistics (Heads / Tails counter)  
- 🔊 Sound effect on each flip  
- 🔁 Reset functionality  

---

## 🧠 Technologies Used

- HTML5  
- CSS3 (Animations & Styling)  
- JavaScript (DOM Manipulation, Events, Logic)  

---

## 📁 Project Structure

coin-flip-simulator/ │── index.html │── style.css │── script.js │── coin.mp3   (optional)

---

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/ozrnehir/coin-flip-simulator.git

2. Open the project folder

3. Run the project:
- Double click index.html  
OR  
- Use Live Server in VS Code  

---

## 🎮 How to Use

- Click Flip to toss the coin  
- The coin animation will play  
- Result updates automatically  
- Click Reset to clear statistics
